两句话优雅扶墙

  • 系统： Debian 11 bullseye

  • Kernel： 官方最新版服务器内核

点我，一键打包下载所有文件！


请使用root权限登录vps

第一句话：

apt update && apt -t bullseye-backports -y install linux-image-cloud-amd64 && apt -y install wget curl vim git nginx ufw && bash -c "$(curl -L https://github.com/XTLS/Xray-install/raw/main/install-release.sh)" @ install -u root && wget https://go.dev/dl/go1.18.3.linux-amd64.tar.gz && tar -C /usr/local -xzf go1.18.3.linux-amd64.tar.gz && export PATH=$PATH:/usr/local/go/bin && go version && git clone https://github.com/XTLS/Xray-core.git && cd Xray-core && go mod download && CGO_ENABLED=0 go build -o xray -trimpath -ldflags "-s -w -buildid=" ./main && systemctl stop xray && mv xray /usr/local/bin/xray && rm -rf /usr/local/share/xray/geoip.dat && cd /usr/local/share/xray/ && wget https://github.com/Loyalsoldier/v2ray-rules-dat/releases/latest/download/geoip.dat && cd /root && wget https://github.com/cloudreve/Cloudreve/releases/download/3.5.3/cloudreve_3.5.3_linux_amd64.tar.gz && tar -zxvf cloudreve_3.5.3_linux_amd64.tar.gz && chmod +x ./cloudreve && vim /etc/ssh/sshd_config

执行完第一句话，把修改好的文件上传至服务器。

然后，你需要做两件事：

1.手动修改sshd_config，如果不需要私钥登录，输入 :wq 回车直接退出；

2.执行 ./cloudreve ，把密码记录下来。 安全起见，执行完第二句话后，登录你的网站，禁止注册功能。

完成后，输入 reboot 重启VPS。


/root/.ssh/authorized_keys 如果你使用SSH私钥登录的话，需要上传公钥 （自己准备）

/root/ssh/*.crt 存放证书 *.key 存放私钥 （已提供v26-dy.ixigua.com 抖音免流自签证书，最好换成自己的）

/etc/sysctl.conf 开启BBR

/etc/nginx/cond.f/*.conf Nginx配置文件，防止主动探测

/usr/local/etc/xray/config.json Xray配置文件

/usr/lib/systemd/system/cloudreve.service 守护Cloudreve

第二句话：

cd /etc/nginx && rm -rf sites-* && chmod 700 /root/.ssh && chmod 600 /root/.ssh/authorized_keys && systemctl daemon-reload && systemctl enable nginx && systemctl enable cloudreve && systemctl restart ssh && systemctl restart nginx && systemctl restart xray && systemctl restart cloudreve && sysctl -p && ufw allow ssh && ufw allow 443 && ufw allow 80 && ufw allow 1024:65535/udp && ufw reload && ufw enable
